Use machine learning to help learn football!

Formation Scout is a free tool to help its users learn more about football through identifying some base offensive formations. This is done through uploading a broadcast image from a game to the Scout App, from there the image is sent to a trained machine learning image classification model where it identifies the formations and returns an annotated image to you.
